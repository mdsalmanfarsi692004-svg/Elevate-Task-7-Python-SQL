## ğŸ“Š Analysis Steps
1.  **Database Connection:** Connected to `sales_data.db` using the `sqlite3` library.
2.  **Data Creation:** Created a `sales` table and inserted mock transaction data.
3.  **SQL Query:** Executed the following query to calculate Total Quantity and Revenue:
    ```sql
    SELECT 
        product, 
        SUM(quantity) AS total_qty, 
        SUM(quantity * price) AS revenue 
    FROM sales 
    GROUP BY product
    ```
4.  **Visualization:** Created a bar chart using `matplotlib` to compare revenue across products.
